# 🎯 Linear Algebra Visualizer — Column Picture Explorer

> *"Don't just solve linear equations. See them."*

An interactive web application that transforms **Gilbert Strang's Column Picture** from MIT 18.06 into a visual, hands-on learning experience.

Instead of memorizing equations, this project lets you **build intuition** by manipulating vectors, exploring linear combinations, understanding span, and discovering why some systems have solutions while others do not.

Designed for students who learn best by **seeing** mathematics.

---

## ✨ Features

* 📐 Interactive 2D Column Picture visualization
* 🌌 3D vector visualization using Three.js
* 🎛️ Real-time matrix and vector editing
* 🎯 Live linear combination builder
* 📊 Span & Reachability Explorer
* 🧩 Independent vs Dependent column demonstrations
* 🌐 Dimension Explorer (2D → 9D)
* 🧠 Guided Learning Mode
* ❓ Interactive challenges & quizzes
* 📈 Matrix Playground
* ⚡ Smooth animations and modern UI
* 🌙 Dark theme designed for long learning sessions

## 🎓 Why This Project?

Most students first encounter Linear Algebra through equations.

This project flips that perspective.

Instead of asking:

> *"Can I solve Ax = b?"*

it asks:

> *"Can these columns combine to reach b?"*

That single shift in thinking—the **Column Picture**—is one of the most important ideas in Linear Algebra and serves as the conceptual foundation for topics such as:

* Machine Learning
* Computer Graphics
* Robotics
* Data Science
* Optimization
* Artificial Intelligence
* Deep Learning
* Computer Vision
* Scientific Computing

The goal of this project is to make those ideas intuitive through interactive visual exploration.

## 💡 Inspiration

This project is heavily inspired by **Professor Gilbert Strang's** legendary **MIT 18.06 – Linear Algebra** lectures.

His explanation of the **Column Picture** fundamentally changes the way Linear Algebra is understood, shifting the focus from solving equations mechanically to thinking geometrically in terms of vectors and linear combinations.

This project is an independent educational visualization created to help students build intuition from those ideas.

All credit for the original mathematical concepts, teaching approach, and classroom explanations belongs to **Professor Gilbert Strang** and the **MIT 18.06 Linear Algebra** course.

If you haven't watched the lectures, you absolutely should—they're among the best introductions to Linear Algebra ever created.
